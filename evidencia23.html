5.8.- Instalación del entorno LAMP en máquina virtual remota

Fecha: Abril / 2026

Descripción

Conectarse vía terminal remota SSH a la instancia de Azure Server para actualizar repositorios lógicos e instalar de forma nativa el servidor web Apache2, el motor lúdico PHP y la base de datos MySQL.

Resultados Obtenidos

Servidor web LAMP completamente funcional en la dirección IP pública de la instancia. Al consultar la URL en el navegador se renderiza de forma exitosa la página por defecto del servidor Apache.

Reflexión Personal

"Pude configurar todo el stack web lúdico LAMP desde cero usando comandos lógicos, permitiéndome ver el potencial de un servidor dedicado en internet."
